### Key Technical Outcomes

- **IntegrationKit:** Abstraction layer for cloud platforms (Azure, GitHub). Unified auth, tools, UI component registry. Two implementations: `AzureKit` + `GitHubKit`.
- **Action routing:** Canonical `api: action-name` message format. Bidirectional LLM ↔ action handler. Prevents ad-hoc action naming across UI/LLM boundary.
- **Skill resolver:** Extensible LLM prompt injection with `phasePrompts` field. Allows task-scoped domain knowledge (AKS Automatic guidance, GitHub workflow patterns).
- **CORS proxy:** Centralized auth middleware for ARM, GitHub, Pricing APIs. Isolates frontend from CORS, token management, rate limiting.
- **Artifact store:** Singleton instance + dependency injection. Binds to LLM session state. Queryable by tool system.
- **Dispatcher:** Unified action handler replacing ad-hoc event listeners. Integrates auth, validation, artifact updates.
