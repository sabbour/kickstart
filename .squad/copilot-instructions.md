# Copilot Coding Agent — Squad Instructions

This file intentionally avoids duplicating repository-wide Copilot guidance.

## Canonical instructions

Read `.github/copilot-instructions.md` for the authoritative Copilot coding agent instructions (team context, capability self-check, branch naming, worktrees, PR guidelines, and decision recording).

## Squad-specific context

If you are working on Squad-routed work, also consult:

1. `.squad/team.md` for the team roster, member roles, and capability profile.
2. `.squad/routing.md` for work routing rules.
3. `.squad/agents/{member}/charter.md` when the issue has a `squad:{member}` label.
4. `.squad/decisions.md` and `.squad/decisions/inbox/` for project decisions and proposed updates.

Keep `.github/copilot-instructions.md` as the single canonical source for shared agent behavior.
