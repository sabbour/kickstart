# v0.5.0 Sprint Plan

**Sprint dates:** 2026-04-11 — 2026-04-25  
**Milestone:** v0.5.0 (18 issues)  
**Theme:** Component catalog + data binding + validation foundation  
**Target release:** 2026-04-29

---

## Strategic Context

v0.4.0 shipped with core conversation engine (6-phase, hybrid state machine + LLM), basic A2UI rendering, and vanilla JS app layer. v0.5.0 focuses on **three critical gaps** identified during Phase 1 validation:

1. **Component catalog completeness** — 5 missing standard components + 2 custom (FileEditor, CostEstimate)
2. **Data and state binding** — components today render static JSON; no dynamic interpolation or cross-component data flow
3. **Validation foundation** — K8s validation, artifact store, and client-side safeguards must exist before user can deploy

18 issues map cleanly to these pillars. We're being realistic about scope: 7 weeks of calendar time, roughly 4 weeks of sprint capacity accounting for design reviews + testing.
